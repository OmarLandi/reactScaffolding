# ReactScaffolding

# Run development Mode
`$ npm start`


# Run compiled project

* Do this one time, to install package globally:

`$ npm i -g http-server-spa`

* then

```
$ npm run build
$ http-server-spa build/ index.html 30000
```

# Run test

`$ npm test`

# Run lint

`$ npm run lint`

# Prettify the code
`$ npm run prettify`
